# 🎓 EduAccess SA - Built with Claude

## Project Overview

**EduAccess SA** is a comprehensive platform connecting South African students with educational opportunities, designed to democratize access to bursaries and funding while empowering parents to support their children's educational journeys.

---

## ⚡ Quick Facts

- **Built in**: 30 minutes as a team activity
- **AI Assistant Used**: Claude (Anthropic)
- **Status**: Fully functional MVP with all core features
- **Technology**: React 18.3+, React Router 6.26+

## 🎯 Core Features

### For Students 👨‍🎓
- Browse available bursaries and opportunities
- Submit applications and track status
- Set reminders for upcoming deadlines
- View personalized opportunities matched to profile
- Manage academic information

### For Parents 👨‍👧
- Link to child's profile securely
- View all applications with progress tracking
- Monitor upcoming deadlines with urgency alerts
- **Subscribe for notifications** about milestones and deadlines
- Access support resources
- Track child's success rate

### For Admins (Sponsors/Professors) 👔
- Create and manage bursary programs
- Set requirements (minimum marks, fields of study)
- Configure deadlines and award amounts
- Track applications and success rates
- Monitor applicant statistics
- Edit or remove bursaries

### For System Admins 📊
- View platform-wide analytics
- Monitor student growth and trends
- Track application success rates
- See field and regional distributions
- Generate insights and recommendations

---

## 🔄 User Workflows

### Student Journey:
```
Register (Student) → Complete Profile → Browse Opportunities → Apply → Track Applications
```

### Parent Journey:
```
Register (Parent) → Link Child's Account → Monitor Progress → Subscribe to Notifications → Support Child
```

### Admin Journey:
```
Register (Admin) → Add Organization Details → Create Bursaries → View Analytics → Manage Applications
```

## 🎨 Color Scheme

The platform uses a **vibrant, welcoming palette**:

- **Primary Gold** (#D4A574): Bursary, achievement, value
- **Vibrant Green** (#10B981): Growth, success, opportunities
- **Bright Blue** (#3B82F6): Trust, information, action
- **Warm Orange** (#F97316): Energy, urgency, motivation
- **Coral Red** (#EF4444): Alerts, urgent deadlines
- **Sunny Yellow** (#FBBF24): Highlights, achievements
- **Clean White**: Clarity and space
- **Soft Grays**: Secondary elements

This color palette conveys **opportunity, growth, and financial support** - exactly what bursaries represent.
